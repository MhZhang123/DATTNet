# DATTNet<br />


Coder for "DATTNet: a Dual-Attention Transformer-based hybrid network for medical image segmentation"<br />


## 1.Environment<br />
Please prepare an environment with Ubuntu 20.04, PyTorch 1.11.0+cu113 and CUDA 11.3.<br />
